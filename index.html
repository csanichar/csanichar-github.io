Mobile Application for Vehicle Maintenance (Current Project)

What is it?

Currently I am working with a Start-up company called Ansik, to help develop a mobile application for both android and IOS that will be able to read engine codes through a vehicle's engine ECU using an on-board diagnostics device that directly plugs into a car below its dashboard and communicates through bluetooth. The languages being utilized to develop this application is html, css, javascript, and angularJS.

Purpose?

The purpose of this product is to be able to utilize these engine codes to observe how a vehicle's functions are performing, including monitoring the battery, breaks, or antifreeze fluid to name a few. This information will help alert owners of any malfunctions occuring within the car, who can take this information to mechanics to get repair quotes and also give them a better understanding of what kinds of problems are occuring with the vehicle.

3D Visualization of Capacitive Micromachined Ultrasonic Transducer (CMUT) Scanning

Product Developed:

Working in a group with four other classmates, we implemented a method to scan an object using a CMUT device and produce a 3D visualized volume with a high Signal-To-Noise ratio using MATLAB.

Stage 1:

In Stage 1 shown to the right we collected raw data using a CMUT to produce multiple 2D images of size 560x360 pixels using MATLAB.

Stage 2:

In Stage 2 we researched and applied image processing techniques to remove noise and sharpen the image quality. The approach we found most beneficial was a method called bilateral filtering.

Stage 3:

In stage 3, the 2D images that we were able to create were then mapped to the real world coordinate system. Following this we used a linear interpolation technique to to map points from the 2D images to a 3D volume. And in order to differentiate the surroundings from the object, we applied an opacity filter to achieve this.
